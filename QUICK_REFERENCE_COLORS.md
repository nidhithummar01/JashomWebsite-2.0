## Copy-Paste Ready Colors

### Primary Actions (Buttons, CTAs)
```tsx
background: '#0EA5E9'
color: '#FFFFFF'
```

### Headings with Accent
```tsx
<span style={{ color: '#0EA5E9' }}>Highlighted Text</span>
```

### Main Background
```tsx
background: '#0B0F14'
```

### Section Background
```tsx
background: '#111827'
```

### Card Background
```tsx
background: 'rgba(255, 255, 255, 0.05)'
```

### Primary Text
```tsx
color: '#FAFAFA'
```

### Secondary Text
```tsx
color: '#9CA3AF'
```

### Subtle Border
```tsx
border: '1px solid rgba(255, 255, 255, 0.08)'
```

### Active Border
```tsx
border: '1px solid rgba(14, 165, 233, 0.3)'
```

### Glow Effect
```tsx
boxShadow: '0 0 20px rgba(14, 165, 233, 0.6)'
```

### Focus State
```tsx
borderColor: 'rgba(14, 165, 233, 0.5)'
boxShadow: '0 0 20px rgba(14, 165, 233, 0.4)'
```

### Gradient
```tsx
background: 'linear-gradient(135deg, #0EA5E9, #06B6D4)'
```
