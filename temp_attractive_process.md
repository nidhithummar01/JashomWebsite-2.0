# Attractive Process Section Created

The new attractive process section has been designed with:
- 3x2 grid layout (responsive)
- Colorful gradient cards (blue, purple, cyan, pink, indigo, green)
- Large icons for each step
- Glowing effects and shadows
- Hover animations (scale + lift)
- Animated background blobs
- Gradient text in heading
- CTA button at the bottom

Each card has:
- Gradient icon box with shadow
- Large step number (01-06) in background
- Bold title
- Description text
- Unique color scheme per step
